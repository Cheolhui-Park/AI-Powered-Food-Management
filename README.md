# 🥗 AI-Powered Smart Food Management System

> **Sustainability-driven automation using JavaScript and Large Language Models (LLMs).**
> Developed to optimize food portioning and minimize domestic waste through data-driven meal planning.

---

## 📖 Project Overview
This project addresses the global issue of food waste by providing an intelligent management layer for domestic kitchens. I leveraged the **OpenAI API** to create a system that doesn't just track inventory, but actively suggests optimized portion sizes and meal plans based on real-time data.

## 🏆 Key Achievements
* **Decoupled Architecture**: Successfully engineered a robust system with separate `frontend` and `backend` environments for better scalability.
* **Asynchronous API Integration**: Implemented advanced **Asynchronous JavaScript (Async/Await)** to handle real-time communication with the OpenAI API, ensuring a seamless user experience without UI freezing.
* **Environmental Impact**: Targeted a reduction in domestic food waste through precise portioning logic.

## 🛠 Technical Stack
- **Frontend**: HTML5, CSS3, JavaScript (ES6+)
- **Backend**: Node.js
- **AI Integration**: OpenAI API (GPT-4)
- **Version Control**: Git / GitHub
